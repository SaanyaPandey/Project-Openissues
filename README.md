<h1>🚀 OpenIssues</h1>

<p><b>An AI-driven platform that simplifies GitHub issue management</b></p>


</div>

<hr/>

<h2>📌 Problem Statement</h2>
<p>
Managing GitHub issues manually becomes overwhelming in large repositories. 
Sorting, prioritizing, and identifying duplicate issues takes significant time and effort.
</p>

<h2>💡 Our Solution</h2>
<p>
<b>OpenIssues</b> is an AI-powered platform that automates issue analysis, 
helping developers focus on resolving problems instead of managing them.
</p>

<hr/>

<h2>⚙️ Key Features</h2>

<ul>
<li>🔍 <b>Repository Analysis</b> – Enter GitHub ID & repo to get issue insights</li>
<li>⚡ <b>Priority Detection</b> – Automatically ranks issues based on importance</li>
<li>🧠 <b>AI-Based Grouping</b> – Categorizes similar issues together</li>
<li>🔁 <b>Duplicate Detection</b> – Identifies and groups repeated issues</li>
<li>🏷️ <b>Unlabeled Issue Sorting</b> – Organizes issues without labels</li>
<li>📊 <b>Insights Dashboard</b> – Shows severity (Low / Moderate / High)</li>
<li>🔐 <b>GitHub OAuth</b> – Connect for real-time data</li>
</ul>

<hr/>

<h2>🖥️ How It Works</h2>

<ol>
<li>Enter your GitHub username and repository</li>
<li>Fetch all issues using GitHub API</li>
<li>AI analyzes and categorizes issues</li>
<li>View grouped, prioritized, and duplicate issues</li>
</ol>
